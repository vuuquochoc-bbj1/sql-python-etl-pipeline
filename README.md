# SQL-to-Python Data Pipeline

A sanitized version of a data pipeline I built during my Grade 11 data science internship. It extracts transactional data from a database, processes it using Python, and prepares it for analytics dashboards.

## 🛠️ Tech Stack
* **SQL Server** (Data Extraction)
* **Python** (Pandas & NumPy for Data Transformation)
* **Power BI** (Data Visualization)

## 📂 Project Structure
* `/sql` - Database queries
* `/scripts` - Python ETL scripts
* `/data` - Mock dataset for testing
